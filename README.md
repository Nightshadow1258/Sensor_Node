# Sensor Node Version 2.00
This is the second version of my sensor node project. This project was inspired by the TrigBoardV7 (https://www.kevindarrah.com/wiki/index.php?title=TrigBoardV7).

## Components
- ESP32-WROOM32 (MCU)
- TLP5110
- TPS63001 (Buck-Converter)
- HDC-1080 (Temp & Humidity Sensor)
